Interactive Reporting – Sample Code


// Example: Compute YoY growth and margin
function computeMetrics(current, previous) {
  const revenueGrowth = ((current.revenue - previous.revenue) / previous.revenue) * 100;
  const netMargin = (current.netIncome / current.revenue) * 100;

  return {
    revenueGrowth: revenueGrowth.toFixed(2) + '%',
    netMargin: netMargin.toFixed(2) + '%'
  };
}

const q4FY25 = { revenue: 5600, netIncome: 842 };
const q4FY24 = { revenue: 4875, netIncome: 730 };

console.log(computeMetrics(q4FY25, q4FY24));

Key Financial Ratios (Math Support)

**Net Profit Margin**: \[ \text{Net Profit Margin} = \frac{\text{Net Income}}{\text{Revenue}} \times 100\% \]

**Return on Equity (ROE)**: \[ \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \times 100\% \]

**Earnings per Share (EPS)**: \[ \text{EPS} = \frac{\text{Net Income} - \text{Preferred Dividends}}{\text{Weighted Avg. Shares Outstanding}} \]

Walk through each formula and relate it to Q4 numbers. Emphasise how these ratios help stakeholders evaluate profitability, efficiency, and shareholder value creation.
